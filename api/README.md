# API (Vercel) - Next steps

Ce projet est actuellement **Vite + React**.

Pour répondre à « AVEC API sur Vercel », il faut ajouter un backend (Serverless) soit via :
- **Next.js** (et donc convertir le repo), soit
- **Vercel Functions** / une approche serverless dans un repo compatible.

Aucune implémentation de fonction n'a été ajoutée automatiquement car elle nécessite un choix d'architecture.

## Prochaine étape
Dès que tu confirmes l'option voulue (1 = convertir en Next.js), je vais :
1) créer le scaffold Next.js + pages/api
2) déplacer ton front Vite/React vers Next.js (ou reconfigurer l’app)
3) brancher le formulaire Contact sur l’endpoint API
